Easily assign different shipping methods to products in WooCommerce and add local pickup addresses.

== Description ==

WooCommerce Shipping Methods For Products is a custom plugin that allows you to assign different shipping methods to individual products and add local pickup addresses. It provides you with the flexibility to manage shipping options for each product and display pickup addresses on customer order confirmation emails.

Features:
* Assign specific shipping methods to individual products.
* Filter available shipping methods in the cart based on selected methods for each product.
* Add a pickup address field to the local pickup shipping method settings.
* Display the pickup addresses in the customer order confirmation email.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-shipping-methods-for-products` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the product edit page, and you will find a new meta box named "Shipping Methods" in the sidebar.
4. Select the desired shipping methods for the product, and save your changes.

== Frequently Asked Questions ==

= Can I use this plugin without WooCommerce? =

No, this plugin requires WooCommerce to be installed and activated, as it extends WooCommerce's shipping functionality.

= Can I set shipping methods for product variations? =

Currently, the plugin only supports setting shipping methods for simple products. Support for product variations may be added in future updates.

= Will this plugin work with other shipping plugins? =

This plugin is designed to work with WooCommerce's built-in shipping methods and should also be compatible with most shipping plugins that follow WooCommerce's coding standards.
